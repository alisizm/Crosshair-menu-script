## 🇬🇧 English Description

### 📖 Overview
Alisizm Crosshair is built for both roleplay and PvP servers, offering a modern and user-friendly crosshair customization experience. Players can easily open the menu via a command, switch between predefined crosshair images, change colors, adjust size, and optionally use a custom crosshair URL.

The script stores player preferences locally using KVP, ensuring each player keeps their personalized settings between sessions. Crosshair visibility is fully configurable: you can choose to show it only while aiming or have it always visible.

### ✨ Key Features
*   **In-Game Menu:** Fast and modern NUI-based crosshair customization menu.
*   **Crosshair Options:** Built-in crosshair list + custom image URL support.
*   **Full Control:** Complete color and size adjustment controls.
*   **Persistent Settings:** Player preferences are saved persistently via local KVP.
*   **Visibility Modes:** Configurable behavior (aim-only or always-visible modes).
*   **Easy Expansion:** Simple to add new crosshairs or adjust existing ones via the config.
*   **High Performance:** Lightweight and optimized behavior for 0 impact on server performance.
*   **Security:** Built-in resource name protection (script won't run under a different name).

### 🛠️ For Server Owners
*   Designed for incredibly quick setup and practical usage.
*   Features a modern, sleek interface that players will love.
*   Highly scalable structure—expanding the asset list is extremely easy.
*   A professional, Tebex-ready package tailored for FiveM communities.

### 🚀 Installation Notes
1.  Place the script folder inside your `resources` directory.
2.  Keep the resource name strictly as `alisizm-crosshair`.
3.  Add `ensure alisizm-crosshair` to your `server.cfg`.
4.  Configure settings in `config.lua` based on your server's specific needs.
